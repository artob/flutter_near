## Prerequisites

- [Dart](https://dart.dev) 2.8+ and [Flutter](https://flutter.dev) 1.17+

## Compatibility

Android and iOS.

## Installation

### `pubspec.yaml`

```yaml
dependencies:
  near: ^0.0.0
```

## Examples

### Importing the library

```dart
import 'package:near/near.dart';
```
